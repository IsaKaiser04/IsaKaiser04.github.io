Descripción del proyecto

es un innovador brazo robótico diseñado para recoger basura en parques, playas y otras áreas públicas. Este brazo es controlado mediante una aplicación desarrollada en AppInventor, que se comunica con Friendly a través de Bluetooth. La aplicación permite a los usuarios mover el brazo robótico, abrir y cerrar la pinza para recoger basura, y programar rutas de limpieza automática.

El proyecto consiste en la creación de un brazo robótico que pueda ser controlado por una aplicación móvil, por medio de conexión Wi-Fi y Bluetooth para su control.
